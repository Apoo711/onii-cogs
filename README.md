 # Installation
To add cogs from this repo to your instance, do these steps:
- `[p]repo add onii-cogs https://github.com/Onii-Chan-Discord/onii-cogs`
- `[p]cog install onii-cogs <cog name>`
- `[p]load <cog name>`
